# Visualizador e Simulador de Grafos

Uma aplicação interativa desenvolvida em Python (Dash/Cytoscape) para criar, editar e simular algoritmos em grafos (Busca em Largura - BFS e Busca em Profundidade - DFS).

## Como rodar o projeto localmente

Para rodar este projeto no seu computador, você precisará ter o **Python** instalado.

### Passo 1: Baixar o projeto
Você pode baixar o código em formato ZIP (botão verde "Code" > "Download ZIP" no topo da página) e extrair no seu computador, ou clonar o repositório via terminal:

```bash
git clone [https://github.com/Richardjss/Visualizador-de-Grafos.git](https://github.com/Richardjss/Visualizador-de-Grafos.git)
```

### Passo 2: Instalar as dependências
Abra o terminal (ou prompt de comando) dentro da pasta do projeto e instale as bibliotecas necessárias rodando o comando:

pip install -r requirements.txt

### Passo 3: Rodar a aplicação
Ainda no terminal, execute o arquivo principal:

python app.py

O terminal mostrará um endereço (geralmente http://127.0.0.1:8050/). Basta copiar e colar esse link no seu navegador para usar o sistema!
